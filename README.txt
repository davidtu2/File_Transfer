How to execute this program:
Pre-requisite(s): sender.cpp and recv.cpp are complied using the Makefile
1. Execute the sender program using picture.jpg as an argument in one console
2. Execute the recv program in another console. After some time, you should see the total number of bytes transferred in both consoles
Also, picture.jpg__recv file should be transferred into the directory
3. Use Ctrl + C within the recv program to deallocate the shared memory and message queue. This should exit the program
